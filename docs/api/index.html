Auth

Register a new user account

Create a new FocusFlow account.

  • Password must be at least 8 characters.
  • Returns a JWT access token and the new user's profile.
  • onboarding_completed will be false — redirect the client to the onboarding screen until PATCH /me/onboarding is called.

required

required
string (Password) [ 8 .. 128 ] characters

Update Block

Update a time block.

scope=this (default) Update only this block with the exact payload supplied.

scope=this_and_future Update this block AND every subsequent block in the same recurrence_group_id series (ordered by start_time ASC). Each following block keeps its own DATE but its TIME is updated to match the edited block's time, preserving the duration. Title, color, task_id, recurrence, and recurrence_group_id are copied verbatim to all affected blocks.
